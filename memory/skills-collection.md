# Skills 收集任务

## 任务说明
定期收集 OpenClaw/AutoX 相关的 Skills，特别是：
- 📄 文档写作（PPT 生成、Markdown、PDF）
- 📊 股票交易（量化、行情、交易 API）
- 🤖 自动化工具
- 🧠 AI 工具集成

## 数据来源
1. **ClawHub** - https://clawhub.ai/
2. **GitHub** - 搜索 "openclaw skill" / "autox skill"
3. **OpenClaw 官方文档** - https://docs.openclaw.ai/
4. **社区推荐** - Discord/论坛

## 更新流程
1. 搜索新 Skills
2. 验证可用性
3. 添加到《OpenClaw Skills 发现库》多维表格
4. 标记分类和推荐指数

## 更新频率
- **每周一次**：搜索新 Skills
- **每月一次**：验证现有 Skills 状态
- **随时**：用户请求特定类型 Skill 时
