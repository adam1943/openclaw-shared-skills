```json
{
  "name": "stock-morning-alert",
  "schedule": "25 9 * * 1-5",
  "command": "查询重点股票集合竞价，推送隔夜新闻和今日策略",
  "enabled": true
}
```

## 推送内容模板

### 早盘推送模板
```
📈 股票早班车 [日期]

【隔夜新闻】
- 新闻 1
- 新闻 2

【集合竞价】
- 永辉超市：X.XX 元 (X.XX%)
- 安泰集团：X.XX 元 (X.XX%)

【今日策略】
- 永辉超市：关注 4.05 元支撑
- 安泰集团：关注 3.55 元支撑

【重点关注】
- 零售板块是否有资金回流
```

### 收盘推送模板
```
📊 股票收盘总结 [日期]

【今日表现】
| 股票 | 收盘价 | 涨跌 | 成交量 |
|------|--------|------|--------|
| 永辉超市 | X.XX | X.XX% | XXX 万 |
| 安泰集团 | X.XX | X.XX% | XXX 万 |

【资金流向】
- 永辉超市：X.XX 亿 (净流入/流出)
- 安泰集团：X.XX 亿 (净流入/流出)

【买卖点提醒】
- 永辉超市：接近买入价 4.05 元，可关注
- 安泰集团：未触及买卖点

【明日策略】
- 永辉超市：关注 4.20 元压力位
- 安泰集团：关注 3.80 元压力位
```

## 监控表格更新

每次推送后自动更新：
1. 股票监控表格 - 最新价格和涨跌幅
2. 行业舆情表格 - 资金流向和舆情热度
3. 记录推送时间和内容
