# PPT Generator Skill - 使用指南

## 📊 Skill 位置
`~/.openclaw/skills/ppt-generator/`

## ✅ 功能特性

### 1. Markdown 转 PPT
- 自动识别标题层级
- 支持列表、代码块、表格
- 生成封面页和目录页

### 2. 飞书文档转 PPT
- 读取飞书云文档
- 保持文档结构
- 支持图片和表格

### 3. 主题模板
| 模板 | 风格 | 适用场景 |
|------|------|---------|
| **business** | 商务深蓝 | 汇报、路演 |
| **tech** | 科技蓝黑 | 技术分享、产品发布 |
| **simple** | 简约黑白灰 | 内部会议、培训 |

### 4. 自动生成
- 封面页（标题 + 日期）
- 目录页（自动提取章节）
- 内容页（多页）
- 总结页

---

## 💡 使用方法

### 基础用法

**对话式生成：**
```
把这篇文档转成 PPT
https://feishu.cn/docx/xxx

生成 PPT，用商务风格
内容：# 标题
- 要点 1
- 要点 2

创建幻灯片
主题：科技
标题：季度报告
内容：...
```

**命令行用法：**
```bash
# Markdown 转 PPT
python generate_ppt.py markdown input.md

# 飞书文档转 PPT
python generate_ppt.py feishu doc_token_xxx

# 文本转 PPT
python generate_ppt.py text "# 标题\n- 要点 1\n- 要点 2"
```

## ⚠️ 限制说明

| 功能 | 支持情况 | 说明 |
|------|---------|------|
| 文字内容 | ✅ 完全支持 | 标题、列表、段落 |
| 图片 | ⚠️ 部分支持 | 需要手动添加 |
| 表格 | ⚠️ 部分支持 | 简单表格支持 |
| 图表 | ❌ 不支持 | 需手动添加 |
| 动画 | ❌ 不支持 | PPT 生成后手动添加 |
| 主题切换 | ✅ 完全支持 | 3 种预设主题 |
