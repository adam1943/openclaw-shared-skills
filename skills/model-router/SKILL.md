# Model Router Skill v0.1

## 概述

智能模型路由系统，根据问题类型自动选择最优模型。

---

## 工作流程

### State 1: 接收问题

**触发条件：** 用户提问

**操作：**
1. 分析问题内容
2. 提取关键词
3. 判断问题类型

---

### State 2: 问题分类

**分类规则：**

| 类型 | 关键词 | 推荐模型 |
|------|--------|---------|
| **代码编程** | 代码、编程、python、js、bug、debug | qwen3-coder-plus |
| **逻辑推理** | 推理、分析、为什么、原因、逻辑 | qwen3-max |
| **文档写作** | 写文章、报告、总结、邮件 | kimi-k2.5 |
| **数据分析** | 数据、统计、图表、excel | qwen3.5-plus |
| **创意写作** | 故事、创意、诗歌、小说 | MiniMax-M2.5 |
| **翻译** | 翻译、英文、中文 | glm-5 |
| **日常对话** | 其他 | qwen3.5-plus (默认) |

---

### State 3: 模型选择

**选择逻辑：**
```python
if "代码" in question or "python" in question:
    model = "qwen3-coder-plus"
elif "推理" in question or "分析" in question:
    model = "qwen3-max"
elif "写" in question or "文章" in question:
    model = "kimi-k2.5"
else:
    model = "qwen3.5-plus"  # 默认
```

---

### State 4: 发送请求

**操作：**
1. 使用选定模型调用 API
2. 记录请求时间
3. 获取响应

---

### State 5: 结果评估

**评估维度：**
- 响应时间（ms）
- 回答质量（1-5 分，用户评分）
- 任务完成度（是否解决问题）

---

### State 6: 数据记录

**记录到 Bitable：**
- 问题类型
- 使用模型
- 响应时间
- 用户评分
- 时间戳

## 数据收集

### Bitable 字段设计

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 问题类型 | 单选 | 代码/推理/写作/分析/对话 |
| 使用模型 | 文本 | 实际使用的模型 |
| 问题内容 | 文本 | 用户原始问题 |
| 响应时间 | 数字 | 毫秒 |
| 用户评分 | 单选 | 1-5 星 |
| 时间戳 | 日期时间 | 自动记录 |
