# Web Search Skill - 免费网页搜索

无需 API 密钥的网页搜索技能，使用浏览器自动化和网页抓取实现搜索功能。

## 功能

- 🔍 搜索任意关键词
- 📰 获取搜索结果摘要
- 🌐 提取网页内容
- 📄 支持多个搜索引擎

## 使用方法

直接告诉我要搜索的内容，例如：
- "搜索周杰伦最新专辑"
- "帮我查一下今天的新闻"
- "搜索 Python 教程"

## 技术实现

使用 `web_fetch` 工具抓取搜索引擎结果页面，无需 API 密钥。

## 支持的搜索引擎

- Google（需要科学上网）
- Bing（推荐，国内可用）
- 百度
- DuckDuckGo

## 限制

- 搜索结果数量：每次最多 10 条
- 页面内容提取：最多 5000 字符
- 依赖网络连接
