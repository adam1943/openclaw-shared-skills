# Web Search Skill

## 安装

技能已创建在：`~/skills/web-search/`

## 使用示例

### 直接搜索
```
搜索周杰伦 2026 新专辑
```

### 搜索特定网站
```
site:zhihu.com Python 入门教程
```

### 搜索新闻
```
2026 年最新 AI 新闻
```

## 工作原理

1. 接收搜索关键词
2. 使用 Bing 搜索引擎
3. 抓取搜索结果页面
4. 提取并返回结果摘要

## 自定义

编辑 `search.ps1` 可以：
- 更改默认搜索引擎
- 调整结果数量
- 添加自定义解析逻辑
